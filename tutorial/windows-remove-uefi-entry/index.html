Tutorial: windows-remove-uefi-entry

Remove the Ubuntu UEFI entry within the Windows UEFI boot partition Start an administration console window

Steps:

  • diskpart
  • list disk # list drives
  • sel disk 1 # select drive number with uefi boot partition
  • list vol # list partitions of drive
  • sel vol 2 # select uefi partition
  • assign letter=U: # assign dirve letter
  • exit # exit diskpart
  • cd /d U:
  • cd EFI
  • ls # double check if ubuntu entry is present
  • rmdir /s ubuntu
  • exit # exit console

Reboot your pc - it will now default boot to windows now.
