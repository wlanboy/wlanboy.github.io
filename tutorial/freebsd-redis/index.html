Tutorial: freebsd redis

Simple way to install redis server on your FreeBSD machine:

  1. Install redis

    cd /usr/ports/databases/redis/ && make install clean

  2. Enable redis service

    sysrc redis_enable=yes

  3. Check redis configuration file

    nano /usr/local/etc/redis.conf
Change content:
bind 127.0.0.1

  4. Start redis service

    service redis start

  5. Redis binaries installed

  • ”/usr/local/bin/redis-sentinel” Redis Sentinel executable (monitoring and failover)
  • ”/usr/local/bin/redis-cli” command line interface utility
  • ”/usr/local/bin/redis-server” the Redis Server
  • ”/usr/local/bin/redis-benchmark” Redis benchmarks
