Tutorial: visual-studio-code-webserver

Visual Studio Code as docker container to run VSCode in a browser

Steps:

  • docker pull codercom/code-server
  • mkdir /projects
  • docker run
docker run --name coder -d -p 9000:8080 --restart unless-stopped -v "/projects:/home/coder/project" codercom/code-server --auth none --disable-telemetry
