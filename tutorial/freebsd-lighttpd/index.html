Tutorial: freebsd lighttpd

Simple way to install lighttpd server on your FreeBSD machine:

  1. Install package

    pkg install lighttpd

  2. Enable lighttpd service

    sysrc lighttpd_enable=yes

  3. Start lighttpd service

    service lighttpd start

  4. Put your files in this directory

    /usr/local/www/data/

  5. Change lighttpd configuration

    nano /usr/local/etc/lighttpd.conf
